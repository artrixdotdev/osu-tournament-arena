# @ota/svelte-config

Shared SvelteKit configuration for the monorepo.

## Usage

Import in your `svelte.config.js`:

```js
import { svelteConfig } from "@ota/svelte-config";

export default svelteConfig;
```

## Included

- `@sveltejs/adapter-cloudflare` - Cloudflare deployment adapter
- `@sveltejs/vite-plugin-svelte` - Vite plugin configuration
