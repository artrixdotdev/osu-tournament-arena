# @ota/prettier-config

Shared Prettier configuration for the monorepo.

## Usage

Add to your `package.json`:

```json
{
   "prettier": "@ota/prettier-config"
}
```

## Included Plugins

- `prettier-plugin-svelte` - Svelte formatting
- `prettier-plugin-tailwindcss` - Tailwind class sorting
- `@ianvs/prettier-plugin-sort-imports` - Import sorting
